## Локально

- Открой `index.html` в браузере  
- или: `python3 -m http.server 8080` → `http://localhost:8080`

## Деплой на GitHub (GitHub Pages)

1. Закоммитьте и запушьте всё в репозиторий на GitHub.
2. В репозитории: **Settings** → **Pages**.
3. **Source**: Deploy from a branch.
4. **Branch**: `main` (или `master`), папка **/ (root)**.
5. Сохраните. Через 1–2 минуты сайт будет доступен по адресу вида `https://<username>.github.io/<repo>/`.

Если репозиторий называется `grumok`, URL будет: `https://<username>.github.io/grumok/`.
